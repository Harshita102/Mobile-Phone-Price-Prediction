# 📱Mobile-Phone-Price-Prediction
## 📌 Project Overview

This project aims to predict mobile phone prices using Machine Learning techniques based on various smartphone specifications. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model development, and performance evaluation to identify the most accurate prediction model.Machine Learning project for predicting mobile phone prices using Linear Regression and Random Forest Regressor. Includes data preprocessing, EDA, feature engineering, model evaluation (MAE, MSE, RMSE, R²), feature importance analysis, and business insights for data-driven price prediction.

---

## 🎯 Objectives

- Analyze a mobile phone dataset containing **541 records**.
- Perform data preprocessing and feature engineering.
- Conduct Exploratory Data Analysis (EDA) to identify patterns and relationships.
- Build and compare **Linear Regression** and **Random Forest Regressor** models.
- Evaluate model performance using **MAE, MSE, RMSE, and R² Score**.
- Identify the most important features influencing mobile phone prices.
- Support data-driven pricing decisions through predictive analytics.
